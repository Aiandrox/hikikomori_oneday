### 操作法

名前と起床時刻を記入したら、「起床ボタン」をクリックするだけ。
あとは、思うままに家でごろごろしよう。
１日を終えた後には、過ごし方についてのコメントもあるよ。

もしかしたら、１日を完遂させられないこともあるかもしれない。人間なので……。


## 使用技術

- Vue.jp(CDN版)
