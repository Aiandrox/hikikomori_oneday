### 操作法

名前と起床時刻を記入したら、「起床ボタン」をクリックするだけ。<br>
あとは、思うままに家でごろごろしよう。<br>
１日を終えた後には、過ごし方についてのコメントもあるよ。

でも、もしかしたら、１日を完遂させられないこともあるかもしれない。人間なので……。
